# Playwright + Pytest UI experiments (SauceDemo)

Небольшой практический репозиторий для экспериментов с UI-автотестами на связке  
**Python + Pytest + Playwright** на примере публичного тестового приложения SauceDemo.

Проект используется как песочница для:
- отработки UI-паттернов
- проверки различных подходов к локаторам и ожиданиям
- экспериментов со структурой тестов
- постепенного наращивания автотестов по мере появления новых задач

---

## Что сейчас реализовано

На текущий момент в репозитории присутствуют:

- базовые UI smoke-тесты
- проверка сценария логина
- минимальная конфигурация Pytest
- использование Playwright `expect` с автоожиданиями
- работа с `data-*` атрибутами как основными локаторами

Проект намеренно не перегружен абстракциями и слоями, чтобы:
- быстрее проверять гипотезы
- проще менять структуру по ходу экспериментов
- не фиксироваться на преждевременной архитектуре
